# Node.js MQTT 客户端使用示例

- 语言：Python3
- 客户端库：[paho-mqtt](https://pypi.org/project/paho-mqtt/)

## 安装依赖

安装 paho-mqtt

使用 pip 安装

```shell script
pip install paho-mqtt
```

或者下载源码安装

```shell script
git clone https://github.com/eclipse/paho.mqtt.python
cd paho.mqtt.python
python setup.py install
```
